Comment utiliser ces composants

  1. Ouvrez le fichier HTML du composant souhaité
  2. Copiez le code HTML du composant
  3. Collez-le dans votre page
  4. Personnalisez les classes Tailwind selon vos besoins
  5. Pour les composants interactifs, assurez-vous d'avoir Alpine.js inclus
